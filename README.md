# Como executar o projeto

```bash
# Clonar repositório
git clone https://github.com/Bruno977/igma.git
```
```bash
# Entrar na pasta do projeto
cd igma
```
```bash
# Instalar Dependências
npm install
```
```bash
# Executar projeto
npm run dev
```
